# Python ile Siber Güvenlik ve Otomasyon Çalışmaları

Bu depo, Python programlama dili kullanılarak geliştirilmiş siber güvenlik araçlarını, ağ scriptlerini, kriptoloji ve API otomasyon çalışmalarımı içermektedir.

---

## 🛠️ Projeler

### 1. P2P Güvenli Mesajlaşma Uygulaması (`p2p_chat.py`)
Python `socket` ve ağ kütüphaneleri kullanılarak geliştirilmiş, sunucuya ihtiyaç duymadan iki cihazın birbiriyle doğrudan haberleşmesini sağlayan bir peer-to-peer uygulamadır.
* **Odak Noktası:** Ağ protokolleri, veri iletimi ve temel kriptoloji/veri güvenliği mantığı.

### 2. API Entegrasyon ve Otomasyon Aracı (`api_araci.py`)
Belirli veri kaynaklarından veya siber güvenlik servislerinden API aracılığıyla dinamik veri çekme, işleme ve otomasyon süreçlerini yöneten script.

---

## 🚀 Çalıştırma

Scriptleri çalıştırmak için ilgili klasörde terminal üzerinden:

```bash
python3 p2p_chat.py
python3 api_araci.py
